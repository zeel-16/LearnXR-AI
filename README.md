# LearnXR AI

LearnXR AI is an open-source AI-powered XR learning platform that combines:

- Computer Vision
- Retrieval Augmented Generation (RAG)
- XR/AR Experiences
- Domain-Specific AI Tutors
- LoRA/QLoRA Fine-Tuning

## Vision

Point your camera at a real-world object and let AI:

- Recognize it
- Explain it
- Teach concepts around it
- Generate quizzes
- Create immersive XR learning experiences
